# Stratagem

Stratagem is an AI-powered FPS performance tracker and game advisor that analyzes how real-life factors (sleep, stress, caffeine, mood) affect competitive gameplay, then turns that data into actionable coaching and personalized improvement plans.

## Tech Stack
- Backend: Python, FastAPI, PostgreSQL
- Frontend: Next.js (React)
- AI: Provider-agnostic system (Gemini, OpenAI-compatible models, and more)

## Current Focus
Building the backend foundation: authentication, session tracking, and the AI system that generates performance insights and improvement advice.
