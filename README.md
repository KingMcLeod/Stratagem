# Stratagem

Stratagem is an AI-powered FPS performance tracker that analyzes how real-life factors
(sleep, stress, caffeine, mood) affect competitive gameplay performance.

## Tech Stack
- Backend: Python, FastAPI, PostgreSQL
- Frontend: React (Next.js planned)
- AI: Gemini (primary), OpenAI, Claude (optional)

## Current Focus
Backend foundation: auth, session tracking, AI advice generation.
